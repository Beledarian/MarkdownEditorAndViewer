# Gemini Markdown Features

This document outlines the special Markdown features supported by this editor, designed to be easily understood and used by AI models like Gemini.

## Colored Text

To apply color to text, use the standard HTML `<span
> tag with the `style` attribute.

### Syntax

```html
<span style="color:COLOR_NAME_OR_CODE;">Your text here</span>
```

### Examples

- **Using a color name:**
  ```html
  <span style="color:red;">This text will be red.</span>
  ```

- **Using a HEX code:**
  ```html
  <span style="color:#00ff00;">This text will be green.</span>
  ```

- **Using an RGB value:**
  ```html
  <span style="color:rgb(0, 0, 255);">This text will be blue.</span>
  ```
