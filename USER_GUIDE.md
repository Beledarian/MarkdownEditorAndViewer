# Markdown Editor - User Guide

Welcome to the **Markdown Editor**, a powerful, local-first tool designed for writers and developers.

### ðŸ“ Editing & Viewing
*   **Split View:** Edit on the left, preview in real-time on the right.
*   **View Mode:** Toggle between split view and full-width preview (`Ctrl+Shift+V`).
*   **Zen Mode:** Press `F11` or click the "Zen" button to hide all UI elements for distraction-free writing.

### ðŸ“‚ File Management
*   **Open Folder:** Click "Show Menu" -> "Open Folder" (`Ctrl+O`) to load a directory.
*   **Sidebar:** Browse files and image assets.
*   **Drag & Drop:**
    *   Drop an `.md` file to open it.
    *   Drop an image to save it to the `assets/` folder and insert it automatically.
*   **Auto-Save:** Enabled by default. Saves changes to the local file system every 2 seconds after typing stops.

### âš¡ Productivity Tools
*   **Daily Word Goal:** Set a target word count in Settings (`Ctrl+,`). The stats bar will turn green when you reach it.
*   **Cheat Sheet:** Click the `?` icon or press `Ctrl+/` for a quick reference of Markdown syntax.
*   **Smart Stats:** Real-time tracking of Words, Characters, Lines, and Reading Time.
*   **Shortcuts:** Fully customizable keyboard shortcuts via the Settings menu.

### ðŸŽ¨ Customization
*   **Themes:** Toggle between Dark (Space Gray) and Light modes (`Ctrl+Shift+T`).
*   **Font Size:** Adjustable editor font size.
*   **Export:**
    *   **HTML:** Download the rendered HTML.
    *   **PDF:** Export the document as a PDF (`Ctrl+Alt+P`).

## Getting Started

1.  **Open the App.**
2.  **Open a Work Folder:** Click "Show Menu" and select your project directory.
3.  **Grant Permission:** Your browser will ask for permission to view and edit files. Click "Verify" or "Allow".
4.  **Start Writing!**

## Tips
*   **Task Lists:** Use `- [ ]` for a checkbox and `- [x]` for a checked box.
*   **Images:** Local images in your project folder will resolve automatically in the preview.
*   **Ignore Files:** You can add patterns (like `node_modules`) to the ignore list in the Sidebar to keep your file list clean.
