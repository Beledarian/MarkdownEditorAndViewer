# User Guide

This guide provides instructions on how to run the Markdown Editor application.

## Running the Application

To run the application, you need to be in the correct directory. All commands must be run from within the `markdown-editor` folder.

1.  **Navigate to the application directory:**
    ```sh
    cd markdown-editor
    ```

2.  **Install dependencies:**
    If you haven't done so already, install the necessary dependencies using npm.
    ```sh
    npm install
    ```

3.  **Start the development server:**
    Run the `dev` script to start the application.
    ```sh
    npm run dev
    ```

After running the `dev` command, the application will be accessible in your web browser at the local address provided in the terminal (usually `http://localhost:5173`).
